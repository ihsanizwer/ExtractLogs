# ExtractLogs
A script used to collect and compress server logs up to 4 times lesser than original size of them.

How to set it up to suit your requirements.
1. Clone it to your management server using git clone https://github.com/ihsanizwer/ExtractLogs.git
2. Change absolute paths to log file location.
3. Change environment names as required. (Should be changed in all applicable places).
4. Grant permissions to execute the script. chmod 755 extractlogs.sh
5. Run the script with required arguments.
